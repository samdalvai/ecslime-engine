# Blob Slash

A JavaScript game for the browser. Built with on an "Entity Component System" game engine.

# How to run

## Prerequisites

- Node.js installed

## Install dependencies

```
npm install
```

## Run

```
npm start
```

Now open the browser at http://localhost:1234

## Run in editor mode

```
npm run start:editor
```

## Clean build files

```
npm run clean
```
